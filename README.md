# Weather application :
**This program is a tool for viewing the weather of cities and displaying all the countries and cities in the world.**
<br>
<br>
**Guidance : **
<br>
<br>
**To install the required libraries for this program, you can use the pip command. Run the following commands in your terminal or command line: **
**Install NTP client for time server communication: pip install ntplib**
**Install requests for HTTP requests: pip install requests**
**Install pycountry for working with country codes: pip install pycountry**
**Install GeonamesCache for geographical data: pip install geonamescache**
