برای نصب کتابخانه‌های مورد نیاز این برنامه، می‌توانید از دستور pip استفاده کنید. در ترمینال یا خط فرمان، دستورات زیر را اجرا کنید:

# نصب NTP client برای ارتباط با سرورهای زمان
pip install ntplib

# نصب requests برای درخواست‌های HTTP
pip install requests

# نصب pycountry برای کار با کدهای کشورها
pip install pycountry

# نصب GeonamesCache برای داده‌های جغرافیایی
pip install geonamescache
